Bespreking Directory Logan Bogaert door Driss Collart

Structuur

De structuur van de repository is op een zeer logische manier opgebouwd. Men kan er gemakkelijk vinden wat men wilt. Opdrachten zijn in hun eigen folder geplaatst en belangrijke files in een apparte folders.

Volledigheid

Enkele opdrachten werden gemaakt en enkelen niet.

Regelmatigheid

Redelijk regelmatig. Elke week worden een aantal commits (3-4) gepusht. Volgens mij voldoende.

Correctheid

De code is overal juist gevalideert. Er zijn geen cruciale fouten in de code. Elke file word juist geladen.

Bereidheid tot verbetering

Voor enkele files zijn meerdere commits gemaakt waar fouten werden verbetert. Dit is dus in orde.
